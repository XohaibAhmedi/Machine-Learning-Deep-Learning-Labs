# Machine-Learning-Deep-Learning-Labs

## 📌 Overview

This repository contains laboratory experiments, assignments, and practical implementations completed during the **8th Semester Machine Learning and Deep Learning Lab course**. It is designed to provide hands-on experience with fundamental and advanced Machine Learning (ML) and Deep Learning (DL) concepts using Python and modern data science frameworks.

The repository covers the complete ML/DL workflow, including data preprocessing, feature engineering, data visualization, machine learning model development, optimization techniques, and deep learning architectures for computer vision and time-series applications.

---

## 📚 Course Coverage

The practical experiments include:

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature engineering and transformation
* Data visualization techniques
* Regression algorithms
* Gradient descent optimization
* Overfitting and underfitting analysis
* Image dataset preparation
* Convolutional Neural Networks (CNNs)
* Data augmentation techniques
* Multi-Layer Perceptrons (MLPs)
* One-Dimensional CNNs
* Long Short-Term Memory (LSTM) Networks
* Time-series forecasting and sequence modeling

---
